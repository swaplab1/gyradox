## Description
GyradoX is an AI-powered Security Operations Center (SOC) investigation and threat intelligence platform designed to streamline incident response, IOC enrichment, and threat hunting workflows. It consolidates data from multiple threat intelligence sources into a single interface, enabling security analysts to investigate indicators of compromise (IOCs) more efficiently and reduce manual analysis time.

The platform integrates real-time threat intelligence feeds, malware reputation services, and OSINT data to provide comprehensive context for IP addresses, domains, URLs, file hashes, and other security artifacts. Its built-in AI-powered "Ask AI" capability analyzes investigation results and delivers contextual insights, threat assessments, and actionable recommendations, helping analysts make faster and more informed decisions.

GyradoX also features an intelligent KQL Query Builder that automatically generates Microsoft Sentinel Kusto Query Language (KQL) queries based on user-provided IOCs, such as IP addresses, domains, URLs, Hashs. This enables analysts to quickly create accurate hunting queries without manually writing KQL, significantly improving the speed and consistency of threat hunting and IOC validation.
